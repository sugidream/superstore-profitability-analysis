# superstore-profitability-analysis
End-to-end Data Analytics and Machine Learning project analyzing Superstore sales data with data cleaning, EDA, feature engineering, classification, regression, and business recommendations.
# Superstore Profitability Analysis

## Project Overview
This project analyzes Superstore sales data and builds machine learning models to predict:

1. Loss-making orders
2. Profit amount

## Skills Demonstrated

- Data Cleaning
- Missing Value Handling
- Duplicate Removal
- Feature Engineering
- Exploratory Data Analysis
- Data Visualization
- Classification Modeling
- Regression Modeling
- Cross Validation
- Feature Importance Analysis

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Business Recommendations

- Limit discounts above 20%
- Improve Furniture profitability
- Focus on high-margin products
- Implement discount approval rules

## Models

### Classification
Random Forest Classifier

### Regression
Random Forest Regressor
